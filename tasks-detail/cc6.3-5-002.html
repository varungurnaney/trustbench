Task

You are a SOC 2 auditor evaluating NovaBridge Technologies' contractor data access controls under CC6.3. NovaBridge engages contractors from several consulting firms for product development and data analytics projects. Review all evidence to assess whether contractor data access is appropriately authorized, time-limited, and revoked upon project completion. Several situations require judgment about whether access persistence is acceptable when NDAs are in place but projects have concluded.

Evidence

  • evidence/data-access-policy.md — NovaBridge Technologies data access policy with contractor access requirements and NDA framework
  • evidence/contractor-access-register.csv — Active contractor access register showing all current contractor data access grants with NDA status, SOW references, and access scope
  • evidence/database-access-log-q4.csv — Q4 2025 database access audit log showing query activity by contractors
  • evidence/nda-register.csv — NDA register showing executed NDAs for all contractors with signature dates and coverage scope
