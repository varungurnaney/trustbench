Task

You are conducting a SOC 2 Type II audit of Zenith Cloud Technologies for the Q4 2025 observation period (October 1 - December 31, 2025). Review all evidence related to their vulnerability management program and assess CC7.1 compliance. Examine scan execution patterns, remediation SLA adherence, risk acceptance practices, and exception management. Pay attention to operational details and cross-reference documents to identify both obvious and subtle control gaps. Not all provided documents may be relevant to this control.

Evidence

  • evidence/vulnerability-management-policy.md — Zenith Cloud Technologies vulnerability management policy with explicit prohibition on scan frequency reduction during code freezes
  • evidence/scan-execution-log-q4-2025.csv — Q4 2025 scan execution log showing weekly Qualys VMDR and AWS Inspector scans with timestamps, asset counts, and findings
  • evidence/remediation-sla-tracking-q4.csv — Remediation SLA tracking spreadsheet with 20 vulnerabilities showing SLA status, remediation dates, and compliance
  • evidence/exception-register.csv — Exception register documenting the code freeze and a remediation SLA extension for legacy application
  • evidence/pentest-report-summary.md — Annual penetration test executive summary — offensive security assessment not directly relevant to vulnerability management program controls [noise]
