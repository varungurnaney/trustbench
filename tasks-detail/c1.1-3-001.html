Task

You are conducting a SOC 2 Type II readiness assessment for Crestline Data Systems covering Q4 2025 (October 1 - December 31, 2025). Review all provided evidence artifacts and cross-reference the data classification policy requirements against the actual database encryption status, DLP configuration, and data flow documentation. Identify inconsistencies between stated policy and actual implementation. Focus on whether confidential data is protected as required.

  • evidence/dlp-rule-config.json — DLP rule configuration across email, CASB, endpoint, API gateway, and database activity monitoring
  • evidence/data-flow-diagram.md — Data flow diagram documenting all Restricted data flows including the new GraphQL API launched October 2025
