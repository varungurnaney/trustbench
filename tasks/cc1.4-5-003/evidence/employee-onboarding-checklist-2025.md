## 1. Overview

This report covers onboarding security checklist completion for all new hires in 2025 (January 1 - August 31).

## 2. Onboarding Checklist Requirements

Each new hire must complete the following within 5 business days:

1. Security Awareness + HIPAA Training
2. Acceptable Use Policy signing
3. Equipment acknowledgment
4. Badge activation
5. IT Security orientation
6. PHI handling procedures (PHI access roles only)
