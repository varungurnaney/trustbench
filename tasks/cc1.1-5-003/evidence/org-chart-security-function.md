## CISO Dual Reporting Structure

Victoria Santos operates under a dual reporting structure:

### Administrative Report → CTO (Michael Torres)

The CISO reports to the CTO for:
- Day-to-day operational coordination
- Technology resource allocation
- Infrastructure and tooling decisions
- Headcount and hiring approvals (within approved budget)

### Functional Report → SRGC Chair (Eleanor Walsh, Independent Board Member)

The CISO reports to the SRGC Chair for:
- Security strategy and program direction
- Policy approvals and risk acceptance
- Budget endorsement
- Performance evaluation
- Program effectiveness assessment

### CTO Role Context

Michael Torres (CTO) has been with Wavecrest since 2018. His role is strategic:

**Day-to-day involvement:**
- Technology roadmap and architecture decisions
- Vendor selection and build-vs-buy
- Engineering team growth and organizational design
- Board and investor technology presentations

**Not involved in:**
- Production operations or incident response
- Security tool configuration or rule management
- Code deployment approvals (VP Engineering authority)
- Security exception approvals (CISO authority)
- Security policy decisions (SRGC authority)

### Independence Safeguards

| Safeguard | Status | Evidence |
|-----------|--------|----------|
| CISO compensation set by Board | Yes | Compensation Committee oversees |
| CISO termination requires Board approval | Yes | Charter Section 6.3 |
| CISO has unrestricted Board access | Yes | Direct line to SRGC Chair |
| Security hold authority (deployment block) | Yes | Can only be overridden by SRGC majority |
| Annual executive session assessing CISO | Yes | December SRGC meeting (management excused) |
| CTO override of CISO in 2025 | None | No recorded instances |
| CTO voting on security policies | No | CTO is non-voting SRGC observer |
