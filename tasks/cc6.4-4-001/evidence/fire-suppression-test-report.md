## 1. System Description

Crestline Analytics operates two FM-200 (HFC-227ea) fire suppression systems protecting the following areas:

| System ID | Protected Area | Agent Quantity | Discharge Time |
|-----------|---------------|----------------|----------------|
| FM-SRV-A | Server Room A (480 sq ft) | 125 lbs | 10 seconds |
| FM-SRV-B | Server Room B (320 sq ft) | 85 lbs | 10 seconds |

## 2. Test Scope

Annual inspection and testing per NFPA 2001 (Standard on Clean Agent Fire Extinguishing Systems) and NFPA 72 (National Fire Alarm and Signaling Code).

## 3. Test Results
